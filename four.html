JavaScript Lessons

Lesson 1: Events & Pop-Up Boxes

Events are triggered by mouse clicks, function calls, keyboard strokes, page load, or form submissions. Event handlers ensure functions respond correctly.

Event Triggers: onclick, onmouseover, onload, onsubmit, etc.

Pop-up Boxes:

  • Alert: alert("message")
  • Prompt: prompt("message", "default")
  • Confirm: confirm("message")

Lesson 2: Conditions, Controls & Loops

Conditional Statements: if, if-else, if-else-if, nested if, switch

Perform actions depending on different conditions.

Example: if(score >= 75){document.write("Passed");} else {document.write("Failed");}

Lesson 3: Functions

Functions are self-contained code blocks performing specific tasks. Can be called anywhere in the page. Arguments can be passed for processing.

Use return to output a value from the function.

Lesson 4: Loops

Loops repeat a block of code until a condition is met.

  • For Loop: known iterations for(var i=0;i<10;i++){...}
  • While Loop: condition-based while(condition){...}
  • Do-While Loop: executes once, then repeats while condition true
  • For-In Loop: iterate arrays or objects

Break and continue control loop flow.

Lesson 5: Strings, Arrays, Date & Math Objects

Strings: text objects with properties and methods (length, charAt(), slice())

Arrays: collections of data with methods (push(), pop(), join())

Date Object: manipulate dates (getDate(), getMonth(), toLocaleString())

Math Object: calculations (Math.PI, Math.sqrt(), Math.random())

Lesson 6: Timing & Math Object

Timing functions allow delayed or repeated code execution.

  • setTimeout() – executes code after specified milliseconds
  • clearTimeout() – cancels a timeout

Math object provides constants and functions for calculations.
