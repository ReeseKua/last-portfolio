3rd Quarter - JS Lessons

JavaScript lessons from L1 to L6.

L1: Introduction to JavaScript

Script: a set of instructions connecting components for a task.

JavaScript is a powerful client-side scripting language created by Brendan Eich. It allows interactive web features, was initially called Mocha → LiveScript → JavaScript, and is not related to Java. Case-sensitive and embedded in HTML.

Characteristics: Interpreted, Client-Side, OOP, Scripted Language.

L2: More About JavaScript

JavaScript is client-side, high-level, interpreted, object-oriented. It reacts to events, validates data, creates cookies, and enhances webpages.

Features: Dynamic typing, responsive, can detect user actions, works with CSS to produce DHTML.

Disadvantages: Browser support limits, source code exposed, potential security risks.

L3: JavaScript Syntax

Terms: DOM, Parsing, Document Object Methods (write(), getElementById()), Document Object Properties (bgColor, fgColor).

Syntax Guides: Quotes (' ' or " "), Comments (// or /* */), Concatenation (+).

L4: Variables & Operators

Variables: Store values that may change. Declared with var or implicitly. Local vs global scope.

Data Types: String, Integer, Float, Boolean, Object, Array, Undefined, Null.

Operators: Arithmetic (+, -, *, /, %), Assignment (=, +=, -=, *=, /=, %=), Comparison (==, ===, !=, !==, >, <, >=, <=).

L5: JS Operations & Functions

Operators: Logical (&&, ||), Concatenation (+ for strings), Math operations with assignment.

Functions: Reusable code blocks. Defined using function name(){}. Can return values for further use.

Example: var result = square(5); where square() calculates x²/2.

L6: Getting Started with JS

Use <script></script> to write JavaScript. Can place in head, body, or external file.

Common Properties & Methods: window (location, open, setTimeout), document (write, cookie, fgColor), math (PI, pow), navigator (appName, platform).

Events: Triggered by user actions. Syntax: <tag attribute="function()">
